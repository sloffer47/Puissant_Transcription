# Transcripteur Audio Puissant

Transcription ultra-rapide audio longs (1h+) avec detection automatique des interlocuteurs.

## Installation Rapide

1. Installe les dependances:
pip install -r requirements.txt

2. Configure ton token Hugging Face:
   - Va sur https://huggingface.co/settings/tokens
   - Cree un token
   - Copie .env.example en .env et ajoute ton token

3. Lance application:
streamlit run app.py

## Fonctionnalites

- Transcription Whisper (large-v3)
- Detection automatique des interlocuteurs
- Optimise pour fichiers 1h+
- Export TXT, JSON, SRT
- Interface Streamlit simple

## Formats supportes

MP3, WAV, M4A, FLAC, OGG, WMA
